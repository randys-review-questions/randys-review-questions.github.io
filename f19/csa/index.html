Randy's Review Questions

AP Computer Science A

Based on content taught by Kevin Faggella in the 2019-2020 school year. All programming questions use the Java language.

1. Consider this line of code in Java:

public static int compareNums(int a, int b) {}

This is an example of a(n)...

    Class declaration
    Instance variable
    Method signature
    Accessor
    Mutator

Correct!

Incorrect. The correct answer was: Method signature

2. Consider the following lines of code in Java. Which of the following expressions would return true? Select all that apply.

String s1 = new String("apple");
String s2 = new String("apple");
    s1==s2;
    s1.equals(s2);
    s1.compareTo(s2)<0;
    s1.compareTo(s2)>0;

Correct!

Incorrect. The correct answers were: s1.equals(s2);

3. Which of the following is true about both ArrayLists and Arrays?

    They have a flexible size
    The index of the first element is 0
    They can only hold objects, not primitives
    They require importing a java utility
    They can only contain boolean values

Correct!

Incorrect. The correct answer was: The index of the first element is 0

4. Consider the following line of Java code:

String[][] ticTacToe = new int[3][3].

Which of the following would print the value at the bottom-right corner of the tic tac toe board?

    System.out.println(ticTacToe[0][0]);
    System.out.println(ticTacToe[3][3]);
    System.out.println(ticTacToe[0][3]);
    System.out.println(ticTacToe[3][0]);
    System.out.println(ticTacToe[2][2]);

Correct!

Incorrect. The correct answer was: System.out.println(ticTacToe[2][2]);

5. If a certain class implements an interface, then...

    It must contain code for all methods declared in the interface
    It must call the super constructor
    It must call all methods declared in the interface
    The constructor must contain print statements that print all code written in the interface
    It will lead to an Index out of bounds exception

Correct!

Incorrect. The correct answer was: It must contain code for all methods declared in the interface

6. Consider the following lines of code in Java. It is known that cats have four legs. What would be the contents of the Cat constructor?

public class Animal {
    private String name; private int numLegs;
    public Animal(String nm, int n) { name = nm; numLegs = n; }
}

public class Cat extends Animal {
    private String furColor;
    public Cat(String nm, String color) { /* answer to this question */ }
}
    super(nm, color);
    super(nm, 4);
    super(4, nm);
    super(nm, 4); furColor = color;
    super(nm, color); furColor = n;

Correct!

Incorrect. The correct answer was: super(nm, 4); furColor = color;

7. Consider the following two for loops. Assume that arr is a one-dimensional array of integers. What is the difference between the i in for loop 1 and the i in for loop 2?

for (int i = 0; i<arr.length; i++) {} // for loop 1
for (int i:arr) {} // for loop 2
    For each iteration, the i in For Loop 2 is always 1 greater than the i in For Loop 1
    For each iteration, the i in For Loop 2 is always 1 less than the i in For Loop 1
    i is an integer in For Loop 1; i is a String in For Loop 2
    i refers to the element in For Loop 1; i refers to the index of the element in For Loop 2
    i refers to the index of the element in For Loop 1; i refers to the element in For Loop 2

Correct!

Incorrect. The correct answer was: i refers to the index of the element in For Loop 1; i refers to the element in For Loop 2

8. Which of the following statements involves skipping all remaining lines of code within a loop for that iteration, but then moving on to the next iteration?

    break;
    continue;
    return;
    arr.remove(i);
    arr.add(i);

Correct!

Incorrect. The correct answer was: continue;

9. Which of the following would generate a random integer between 0 and 10, inclusive?

    Math.random()*10
    (int)(Math.random()*10)
    (int)Math.random()*10
    Math.random()*11
    (int)(Math.random()*11)

Correct!

Incorrect. The correct answer was: (int)(Math.random()*11)

10. Which of the following expressions return true? Select two options.

    (3<5) || (8<3)
    (3<5) && (8<3)
    (1+1==2)
    (1+1=2)

Correct!

Incorrect. The correct answers were: (3<5) || (8<3), (1+1==2)
