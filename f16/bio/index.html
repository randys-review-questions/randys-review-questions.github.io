Randy's Review Questions

Biology

Based on content taught by Pam Missal during the 2016-2017 school year.

1. The variable that is being measured in science is the...

    Independent Variable
    Dependent Variable
    Control
    Constant

Correct!

Incorrect. The correct answer was: Dependent Variable

2. A human population pyramid of a developing country is a _____ shape.

    Triangle
    Diamond
    House
    Beehive

Correct!

Incorrect. The correct answer was: Triangle

3. A disaccharide is a sugar with...

    one monosaccharide
    two monosaccharides
    zero monosaccharides
    3+ monosaccharides

Correct!

Incorrect. The correct answer was: two monosaccharides

4. In which organelle are proteins packaged into vesicles?

    Endoplasmic Reticulum
    Nucleus
    Mitochondria
    Golgi bodies

Correct!

Incorrect. The correct answer was: Golgi bodies

5. The code for the sequence of amino acids in protein ultimately comes from...

    DNA
    mRNA
    tRNA
    Ribosome

Correct!

Incorrect. The correct answer was: DNA

6. Sex cells contain a ____ number of chromosomes.

    Diploid
    Triploid
    Haploid
    Quenploid

Correct!

Incorrect. The correct answer was: Haploid

7. Cross HH x hh. What percentage probability is there for an offspring with Hh?

    0%
    25%
    50%
    100%

Correct!

Incorrect. The correct answer was: 100%

8. A bacteria that produces human insulin would be classified as a(n)...

    Human
    Lipid
    Transgenic organism
    Animal

Correct!

Incorrect. The correct answer was: Transgenic organism

9. The _____ undergoes evolution.

    Individual
    Species
    Cell
    Ecosystem

Correct!

Incorrect. The correct answer was: Species

10. Which of the following is both a domain AND a kingdom?

    Eukarya
    Archaea
    Protista
    Animalia

Correct!

Incorrect. The correct answer was: Archaea
