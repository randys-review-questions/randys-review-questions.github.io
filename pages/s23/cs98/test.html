1. When working in sprints, during which meetings should team members briefly state which issues are blocking them from progress?

    Sprint Planning meetings
    Work sessions
    Stand Up meetings
    Sprint Retrospective meetings

Correct!

Incorrect. The correct answer was: Stand Up meetings

2. Ideally, each sprint in a software development project lasts about how long?

    2 days
    1 week
    2 weeks
    1 month

Correct!

Incorrect. The correct answer was: 2 weeks

3. When working on a large software project, there is often a list of criteria that the final product should meet; such criteria may include certain functional, performance, and/or usability requirements. Tests that evaluate whether such criteria are met are considered to be...

    unit tests.
    integration tests.
    regression tests.
    acceptance tests.

Correct!

Incorrect. The correct answer was: acceptance tests.

4. The items discussed in the Sprint Retrospective meeting should be reported in which deliverable?

    Executive Summary
    Acceptance Test Results
    Project Poster
    Final Report

Correct!

Incorrect. The correct answer was: Executive Summary

5. The Digital Design Studio, where one can get their poster printed for the Poster Session, is located in which building on the Tufts campus?

    Mayer Campus Center
    Tisch Library
    Joyce Cummings Center
    Dowling Hall

Correct!

Incorrect. The correct answer was: Tisch Library

6. Which of the following is the correct way to notate a middle C whole note in MusiCode?

    C1/1
    C1/4
    C4/1
    C4/4

Correct!

Incorrect. The correct answer was: C4/1

7. Consider the following MusiCode token. The underscore ("_") character signifies...

C4_G4/4
    a chord.
    a slur between two notes.
    a glissando between two notes.
    a tremolo between two notes.

Correct!

Incorrect. The correct answer was: a chord.

8. Consider the following partial implementation of "Mary Had a Little Lamb" (below is the incomplete MusiCode source and the equivalent PDF). Fill in the blank in the MusiCode source.

(title "Mary Had a Little Lamb")
(composer "Sarah Josepha Hale")
(part "Violin"
  (staff
    (key DM)
    _____
    (barline final)
  )
)
    (notes F4./4 E4/8 D4/4 E4/4 F4/4 F4/4 F4/2)
    (notes F#4./4 E4/8 D4/4 E4/4 F#4/4 F#4/4 F#4/2)

    (notes F#4/4. E4/8 D4/4 E4/4 F#4/4 F#4/4 F#4/2)

Incorrect. The correct answer was: (notes F#4/4. E4/8 D4/4 E4/4 F#4/4 F#4/4 F#4/2)

9. What should the MusiCode notes block look like to produce the following beaming pattern?

    (notes [ C4/8 C4/8 ] [ C4/8 C4/8 ])
    (notes C4/8 [ C4/8 ] C4/8 [ C4/8 ])
    (notes C4/8 C4[beamNone]/8 C4/8 C4[beamNone]/8)
    (notes C4[beamBegin]/8 C4[beamEnd]/8 C4[beamBegin]/8 C4[beamEnd]/8)

Correct!

Incorrect. The correct answer was: (notes C4[beamBegin]/8 C4[beamEnd]/8 C4[beamBegin]/8 C4[beamEnd]/8)

10. All of the following are defined as "staff-level events" in MusiCode EXCEPT...

    Tuplets
    Multiple Voices Blocks
    Multiple Endings (first, second, etc.)

15. In MusiCode, "modifiers" are specified for notes by placing keywords in square brackets between the octave and the duration; such "modifiers" can include dynamic markings, articulation markings, and/or more. In LilyPond, most such "modifiers" are specified via some symbol or token written after the note being modified (in the LilyPond code), but there are a few notable exceptions where the "modifiers" are instead specified before the note being modified. Which of these MusiCode modifiers, when transpiled to LilyPond, ends up being specified before the note being modified (as opposed to after)?

    phrasingSlurBegin
    tremolo8
    decrescEnd
    stemDown

Correct!

Incorrect. The correct answer was: stemDown
