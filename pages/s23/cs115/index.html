Randy's Review Questions

CS 115: Database Systems

Based on content taught by Johes Bater in Spring 2023.

1. Consider a relation with 20 rows and 4 columns. What is the degree of this relation?

    2
    4
    20
    80

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: 4

2. Consider a relation with a schema where every attribute is fully functionally dependent on the candidate key, and one of the fields in each tuple is a set of references to other tuples in the relation. This relation is in...

    first normal form but not necessarily second normal form.
    second normal form but not necessarily third normal form.
    third normal form.
    None of the above

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: None of the above

3. Suppose that relation \(A\) stores information about grocery stores and the fruit sold within each store, and relation \(B\) is just a one column relation of fruits. The schema is defined as follows (the set of underlined attributes for a given relation defines the candidate key):

\(A\)(store_id, store_name, fruit)
\(B\)(fruit)

What information is stored in \(A/B\)?

    The store ID and names of all stores in \(A\) that sell all fruit listed in \(B\)
    The store ID and names of all stores in \(A\) that sell at least one fruit listed in \(B\)
    The store ID, names, and fruit sold for all stores in \(A\) that sell all fruit listed in \(B\)
    The store ID, names, and fruit sold for all stores in \(A\) that sell at least one fruit listed in \(B\)

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: The store ID and names of all stores in \(A\) that sell all fruit listed in \(B\)

4. Consider a relational database of tourists and countries the tourists visited, containing the following schema (the set of underlined attributes for a given relation defines the candidate key):

Tourist(tid, name, age, sex, cid_country_visited, date_of_visit)
Country(cid, name, continent, date_founded)

Which of the following is a valid SQL query of this database?

    SELECT T.name, C.name, MAX(T.age) FROM tourist T JOIN country C ON T.cid_country_visited = C.cid GROUP BY T.name
    SELECT T.name, C.name, MAX(T.age) FROM tourist T JOIN country C ON T.cid_country_visited = C.cid GROUP BY T.name, MAX(T.age)
    SELECT T.name, MAX(T.age) FROM tourist T JOIN country C ON T.cid_country_visited = C.cid GROUP BY T.name, C.name
    SELECT T.name, MAX(T.age) FROM tourist T JOIN country C ON T.cid_country_visited = C.cid GROUP BY T.name, C.name, MAX(T.age)

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: SELECT T.name, MAX(T.age) FROM tourist T JOIN country C ON T.cid_country_visited = C.cid GROUP BY T.name, C.name

5. Which of the following statements regarding database storage is INCORRECT?

    Relying on the operating system to move pages of database information to/from memory results in suboptimal performance.
    In a heap file, tuples are stored in a sorted order (based on some key).
    Physical denormalization of tuple data can improve the performance of common join operations.
    Log structured storage is desirable when anticipating a workload with many writes and few reads.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: In a heap file, tuples are stored in a sorted order (based on some key).

6. Which of the following replacement policies (of frames in the buffer pool) generally results in the highest hit rate when the database's data is accessed multiple times in a looping sequence?

    First Come, First Served
    Least Recently Used
    Most Recently Used
    Clock Replacement

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Most Recently Used

7. When there exists a B+ tree indexing into a relation on a particular search key (and no other index), equality queries on that key take _____ time, and range queries on that key take _____ time.

    constant, logarithmic
    logarithmic, logarithmic
    constant, linear
    logarithmic, linear

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: logarithmic, logarithmic

8. For which of the following hash index implementations are overflow pages not supported?

    Static Hashing
    Extendible Hashing
    Linear Hashing
    Quadratic Hashing

9. Suppose that you are executing a block nested loop join with relation \(A\) as the outer relation and relation \(B\) as the inner relation. \(A\) has 20 tuples and \(B\) has 30 tuples. Additionally, 5 tuples fit on each page, and there is enough space for 4 pages in the buffer pool. What is the I/O cost (in number of pages) of joining \(A\) and \(B\)?

    16
    24
    28
    124

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: 16

10. In which of the following DBMS process models does the DBMS mostly manage its own scheduling?

    Process per Worker
    Thread per Worker
    Application per Worker
    Embedded DBMS

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Thread per Worker

11. Suppose that you are joining relations \(A\) and \(B\) with \(A\) as the outer relation and \(B\) as the inner relation, and \(B\) has a very large number of tuples compared to \(A\). Which of the following join implementations would result in the best performance?

    Block Nested Loop Join
    Index Nested Loop Join
    Sort-Merge Join
    Hash Join

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Index Nested Loop Join

12. Which of the following is NOT a necessary property of a transaction?

    Any changes made by a committed transaction persist, even if the power is disconnected.
    When multiple transactions are interleaved, the end result is the same as if the transactions were executed in a serial order.
    Either all of the actions within a transaction execute or none of them do.
    Consistent transactions, when executed, cause databases in an inconsistent state to end up in a consistent state.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Consistent transactions, when executed, cause databases in an inconsistent state to end up in a consistent state.

13. Suppose that \(T_1\) is a transaction holding lock \(A\) and \(T_2\) is another transaction requesting lock \(A\) (while \(T_1\) is holding it). Which of the following accurately describes what would happen when using the "wait-die" method of deadlock prevention (in two-phase locking)?

    If \(T_1\) is older than \(T_2\), \(T_2\) will wait for \(T_1\); otherwise, \(T_1\) will abort.
    If \(T_1\) is older than \(T_2\), \(T_2\) will wait for \(T_1\); otherwise, \(T_2\) will abort.
    If \(T_1\) is older than \(T_2\), \(T_1\) will abort; otherwise, \(T_2\) will wait for \(T_1\).
    If \(T_1\) is older than \(T_2\), \(T_2\) will abort; otherwise, \(T_2\) will wait for \(T_1\).

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: If \(T_1\) is older than \(T_2\), \(T_2\) will abort; otherwise, \(T_2\) will wait for \(T_1\).

14. Which of the following buffer pool policies does almost every DBMS use, and why?

    NO-FORCE + STEAL because it has the faster runtime performance
    NO-FORCE + STEAL because it has the faster recovery performance
    FORCE + NO-STEAL because it has the faster runtime performance
    FORCE + NO-STEAL because it has the faster recovery performance

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: NO-FORCE + STEAL because it has the faster runtime performance
