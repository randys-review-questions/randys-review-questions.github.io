Randy's Review Questions

CS 105: Programming Languages

Based on content taught by Richard Townsend in Spring 2022. Programming questions may use the Impcore, micro-Scheme, ML, nano-ML, and/or micro-Smalltalk languages, as well as typed versions.

1. Which of these steps comes first in the 9 step design process?

    Write the function's contract
    Write the function's unit tests
    Understand the forms of data
    Write the algebraic laws

Correct!

Incorrect. The correct answer was: Understand the forms of data

2. When writing algebraic laws, the left-hand side defines _____, whereas the right-hand side defines _____.

    the cases, what to do in each case
    what to do in each case, the cases
    looping parameters, what to do in each iteration
    what to do in each iteration, looping parameters

Correct!

Incorrect. The correct answer was: the cases, what to do in each case

3. What proof system for natural numbers defines a natural number as either \(0\) or the successor of another natural number?

    Peano
    Binary
    Decimal
    Parity

Correct!

Incorrect. The correct answer was: Peano

4. Consider the following lines of Impcore. Which of the following correctly shows the contents of the globals and formals environments (\(\xi\) and \(\rho\), respectively) after the lines are executed?

(val x 3)
(set x 4)
(val y 4)
    Globals: \(\{ x \to 3, y \to 4 \}\); Formals: \(\{x \to 4\}\)
    Globals: \(\{ x \to 3, y \to 4 \}\); Formals: \(\{\}\)
    Globals: \(\{ x \to 4, y \to 4 \}\); Formals: \(\{\}\)
    Globals: \(\{ x \to 4\}\); Formals: \(\{ x \to 3, y \to 4\}\)

Correct!

Incorrect. The correct answer was: Globals: \(\{ x \to 4, y \to 4 \}\); Formals: \(\{\}\)

5. Consider the following judgement:

\(\langle e, \xi, \phi, \rho \rangle \Downarrow \langle v, \xi', \phi, \rho' \rangle\)

A prime symbol (\('\)) appears next to \(\xi\) and \(\rho\) on the right side but not \(\phi.\) This signifies that the environments represented by \(\xi\) and \(\rho\) _____ during the process of evaluation, whereas the environment represented by \(\phi\) _____.

    definitely changed, may have stayed the same
    definitely changed, definitely stayed the same
    may have changed, may have stayed the same
    may have changed, definitely stayed the same

Correct!

Incorrect. The correct answer was: may have changed, definitely stayed the same

6. The inference rule for \(LITERAL\) (Impcore) has the following judgement as its conclusion:

\(\langle LITERAL(v), \xi, \phi, \rho \rangle \Downarrow \langle v, \xi, \phi, \rho \rangle\)

What premise(s) is/are necessary to arrive at that conclusion?

    \(v\) is in the domain of \(\xi\).
    \(v\) is in the domain of \(\rho\) AND \(v\) is not in the domain of \(\xi\).
    Either of the above suffices.
    No premises are necessary.

Correct!

Incorrect. The correct answer was: No premises are necessary.

7. Consider the following lines of Impcore. What literal is the variable x mapped to after execution?

(val x 1)
(define func (x y) (set x 4))
(func (set x 2) (set x 3))
    1
    2
    3
    4

Correct!

Incorrect. The correct answer was: 3

8. Which of the following is equivalent to the list '(1 2) in micro-Scheme?

    (cons 1 (cons 2 '()))
    (cons 1 '(2))
    (cdr (cons (cons 1 (cons 2 '())) (cons 1 (cons 2 '()))))
    All of the above

Correct!

Incorrect. The correct answer was: All of the above

9. Consider the following function defined in micro-Scheme. Which of the following successfully returns the sum of 2 and 3?

(define add (y) (lambda (x) (+ x y)))

    (add 2 3)
    (add (2 3))
    ((add 2) 3)
    (((add) 2) 3)

Correct!

Incorrect. The correct answer was: ((add 2) 3)

10. Fill in the blank in the following algebraic laws for the list map function (micro-Scheme).

(map f '())         == '()
(map f (cons x xs)) == _____
    (cons (f x) (map f xs))
    (map (cons (f x) (f xs)))
    (cons (map f x) (f xs))
    (map (f x) (cons f xs))

Correct!

Incorrect. The correct answer was: (cons (f x) (map f xs))

11. Which of the following describes the use of a higher order function?

    A function that passes in a function as a parameter
    A function that returns a function as a result
    All of the above
    None of the above

Correct!

Incorrect. The correct answer was: All of the above

12. What is a continuation?

    A function passed in as a parameter that is applied to all elements of a list.
    A function passed in as a parameter that defines what happens next after some computation occurs.
    A function returned by a function that awaits further input.
    A list traversal function

Correct!

Incorrect. The correct answer was: A function passed in as a parameter that defines what happens next after some computation occurs.

13. A backtracking algorithm is essentially a(n) _____ in the space of possible solutions.

    breadth first search
    depth first search
    topological sort
    single source shortest path search

Correct!

Incorrect. The correct answer was: depth first search

14. What is the type of the following function in ML?

fun func w (x, y) []        = w + x + y
  | func w (x, y) (z :: zs) = z + func w (x, y) zs
    fn : 'a ('b, 'c) ('d list) -> int
    fn : int (int, int) (int list) -> int
    fn : 'a -> 'b * 'c -> 'd list -> int
    fn : int -> int * int -> int list -> int

Correct!

Incorrect. The correct answer was: fn : int -> int * int -> int list -> int

15. Consider the following datatype definition in ML:

datatype int_tree = EMPTY | INSERT of int_tree * int * int_tree

Which of the following is the type constructor?

    int_tree
    INSERT
    int_tree * int * int_tree
    EMPTY | INSERT of int_tree * int * int_tree

Correct!

Incorrect. The correct answer was: int_tree

16. Suppose that we are defining a type env that stores the mapping from a variable name to some value. An inference rule that specifies how to create new values of type env is referred to as a(n)...

    formation rule.
    introduction rule.
    admittance rule.
    elimination rule.

Correct!

Incorrect. The correct answer was: introduction rule.

17. Consider the following abstract syntax definition of type d:

\(CONAPP(TYCON(a), [TYCON(b), TYCON(c)])\)

Which of those types has kind \(*\)?

    a only
    a and d only
    b and c only
    b, c, and d only

Correct!

Incorrect. The correct answer was: b, c, and d only

18. What is the type of '(()), or a one-element list containing the empty list, in typed micro-Scheme?

    (list (list 'a))
    (list (forall ['a] (list 'a)))
    (forall ['a] (list (list 'a)))
    (forall ['a] (list (forall ['a] (list 'a))))

Correct!

Incorrect. The correct answer was: (list (forall ['a] (list 'a)))

19. Consider the following line of nano-ML code:

(lambda (x y z) (if x y z))

Suppose that x has type 'a, y has type 'b, and z has type 'c (each of which are fresh, distinct type variables). What constraint is present as a consequence of this line of code?

    'a ~ bool
    'a ~ bool /\ 'b ~ int /\ 'c ~ int
    'a ~ bool /\ 'b ~ 'c
    'a ~ bool /\ 'a ~ 'b /\ 'b ~ 'c

Correct!

Incorrect. The correct answer was: 'a ~ bool /\ 'b ~ 'c

20. What substitution fully satisfies the following constraint?

'a1 ~ bool /\ 'a2 ~ 'a3 /\ list 'a3 ~ list int

    'a1 |--> bool o 'a3 |--> int
    'a1 |--> bool o 'a2 |--> int o 'a3 |--> int
    'a1 |--> bool o 'a2 |--> (list int) o 'a3 |--> int
    idsubst

Correct!

Incorrect. The correct answer was: 'a1 |--> bool o 'a2 |--> int o 'a3 |--> int

21. What does it mean for a substitution to be idempotent?

    The substitution does not satisfy the constraint.
    The substitution is not the only substitution to satisfy the constraint.
    If the substitution is applied twice, the result is the same as if it were applied once.
    If the substitution is applied twice, the result is not the same as if it were applied once.

Correct!

Incorrect. The correct answer was: If the substitution is applied twice, the result is the same as if it were applied once.

22. What is the ML equivalent of an interface, holding declarations without their implementations?

    Signature
    Structure
    Record
    Functor

Correct!

Incorrect. The correct answer was: Signature

23. Consider the following declaration in micro-Smalltalk:

(val y [block (x) x])

Which of the following correctly utilizes y to return the number 2?

    (y 2)
    (y: 2)
    (y value 2)
    (y value: 2)

Correct!

Incorrect. The correct answer was: (y value: 2)

24. Classes that are meant to be inherited from but not to be instantiated are called...

    abstract classes.
    concrete classes.
    superclasses.
    subclasses.

Correct!

Incorrect. The correct answer was: abstract classes.

25. The use of double dispatch helps avoid what kind of structure in code?

    Conditionals
    Loops
    Recursion
    Dynamic Dispatch

Correct!

Incorrect. The correct answer was: Conditionals
