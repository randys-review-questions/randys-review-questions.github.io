CS 116: Introduction to Security

Based on content taught by Ming Chow in Spring 2022.

1. How does one fully secure an airport?

    Strictly follow the TSA's guidelines.
    Place an officer at every square yard.
    Close the airport permanently.
    It is impossible to fully secure an airport.

Correct!

Incorrect. The correct answer was: Close the airport permanently.

2. What packet is first sent in the TCP three-way handshake?

    SYN
    ACK
    SYN-ACK
    None of the above

Correct!

Incorrect. The correct answer was: SYN

3. What does the linux file command do?

    Creates an empty file
    Outputs the contents of a file
    Outputs the type of a file
    Compresses a file

Correct!

Incorrect. The correct answer was: Outputs the type of a file

4. Which of the following provides a mapping between names such as google.com and their corresponding IP addresses?

    Internet Protocol
    Address Resolution Protocol
    Domain Name System
    Packet Capture

Correct!

Incorrect. The correct answer was: Domain Name System

5. What tool is useful for packet analysis?

    Burp Suite
    Wireshark
    Hashcat
    Nikto

Correct!

Incorrect. The correct answer was: Wireshark

6. What port number is typically associated with the HTTP protocol?

    21
    22
    80
    7777

Correct!

Incorrect. The correct answer was: 80

7. How many layers of abstraction are there in the Open Systems Interconnection (OSI) model?

    5
    6
    7
    9

Correct!

Incorrect. The correct answer was: 7

8. What encryption method is impossible to crack?

    Caesar cipher
    Base64
    One-Time Pad
    No encryption method is impossible to crack.

Correct!

Incorrect. The correct answer was: One-Time Pad

9. Which file in Linux systems stores password hashes?

    /etc/passwd
    /etc/shadow
    /usr/bin/passwd
    /usr/bin/shadow

Correct!

Incorrect. The correct answer was: /etc/shadow

10. What is the difference between a CVE and a CWE?

    A CVE describes a weakness, whereas a CWE describes a mitigation strategy for that weakness.
    A CVE describes a mitigation strategy for a weakness, whereas a CWE describes the weakness itself.
    A CVE describes a generic weakness, whereas a CWE describes a weakness tied to a specific product or service.
    A CVE describes a weakness tied to a specific product or service, whereas a CWE describes a generic weakness.

Correct!

Incorrect. The correct answer was: A CVE describes a weakness tied to a specific product or service, whereas a CWE describes a generic weakness.

11. The buffer overflow vulnerability is present in many functions of the _____ language(s).

    C and C++
    Java and JavaScript
    Python
    Impcore and Scheme

Correct!

Incorrect. The correct answer was: C and C++

12. Consider the following SQL statement that is executed based on a user-inputted username and password:

SELECT username, password FROM users WHERE password = '$password' AND username = '$username'

Which of the following inputs of username/password combos would result in information returned, thereby allowing one to "break in" to the system?

Format: username:password

    blah:1 OR '1'='1'
    blah:1' OR '1'='1
    1 OR '1'='1':blah
    1' OR '1'='1:blah

Correct!

Incorrect. The correct answer was: 1' OR '1'='1:blah

13. Which of the following statements about static/dynamic analysis is incorrect?

    Valgrind is a static analysis tool.
    Static analysis guarantees full code coverage.
    Dynamic analysis involves program execution.
    Linters are not dynamic analysis tools.

Correct!

Incorrect. The correct answer was: Valgrind is a static analysis tool.

14. Which of the following best describes code that self-replicates without requiring any user intervention?

    A proxy
    A virus
    A worm
    A backdoor

Correct!

Incorrect. The correct answer was: A worm

15. Which of the following best describes how anti-virus programs are generally implemented?

    Run each executable in a controlled environment and report whether the program attempts to access sensitive information.
    Test each file on a neural network trained on past examples of viruses and non-viruses, reporting the network's prediction.
    Hash each file and compare those hashes against the hashes of known viruses, reporting if there is a match.
    Wait for some small, random amount of time and report that no virus was detected.

Correct!

Incorrect. The correct answer was: Hash each file and compare those hashes against the hashes of known viruses, reporting if there is a match.
