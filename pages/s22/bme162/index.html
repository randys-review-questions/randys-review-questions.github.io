Randy's Review Questions

BME 162: Molecular Biotechnology

Based on content taught by Niall Lennon in Spring 2022.

1. Which of the following would be an example of "old biotechnology," developed thousands of years ago?

    Recombinant DNA techniques
    Stem cell research
    Synthetic biotechnology
    Beer brewing

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Beer brewing

2. Based on what we know about genetically modified organisms (GMOs), which of the following is a valid concern?

    GMOs are not safe to eat.
    Overuse of herbicides on herbicide-resistant crops leads to more herbicide-resistant weeds.
    Introducing genetically modified crops is bad for the environment.
    None of the above

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Overuse of herbicides on herbicide-resistant crops leads to more herbicide-resistant weeds.

3. In an industrial biotechnology process, the stage where the final product is purified is referred to as...

    fermentation.
    transformation.
    upstream processing.
    downstream processing.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: downstream processing.

4. What is a plasmid?

    A piece of DNA found in bacteria that is not part of the main DNA.
    A section of the main DNA in bacteria.
    The structure holding genetic information in viruses.
    A piece of DNA found in certain organelles of eukaryotic cells.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: A piece of DNA found in bacteria that is not part of the main DNA.

5. What aspect about semi-random mutagenesis makes it "semi-random"?

    The use of PCR.
    The use of degenerate primers.
    The use of circular ssDNA.
    The use of DNA shuffling.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: The use of degenerate primers.

6. When testing for the presence of a virus, a rapid antigen test detects the presence or absence of...

    viral protein.
    viral DNA.
    viral RNA.
    an immune response to viral infection.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: viral protein.

7. Which of the following is a true statement regarding RT-PCR COVID-19 tests?

A qualitative test indicates whether or not you have COVID-19. A quantitative test indicates the amount of viral load you have.

    RT-PCR tests are approved to be used as both qualitative and quantitative tests.
    RT-PCR tests were designed to be used as qualitative and quantitative tests, but are only approved to be used as qualitative tests.
    RT-PCR tests were designed to be used as qualitative and quantitative tests, but are only approved to be used as quantitative tests.
    RT-PCR tests were designed to be used as qualitative tests and were not designed to be used as quantitative tests.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: RT-PCR tests were designed to be used as qualitative and quantitative tests, but are only approved to be used as qualitative tests.

8. Which of the following lists COVID variants from the least to most severe (when disregarding transmissibility)?

    alpha, omicron, delta

10. The process of cloning involves fusing an enucleated egg cell with an adult skin cell, letting that fused cell divide a few times, then planting the resulting embryo in a surrogate mother. The cloned animal will have the same DNA as...

    the animal that produced the enucleated egg cell.
    the animal that fertilized the enucleated egg cell.
    the animal that produced the skin cell.
    the surrogate mother.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: the animal that produced the skin cell.

11. What is one advantage of CRISPR over ZFNs and TALENs?

    CRISPR can edit DNA without introducing a double-stranded break.
    CRISPR can edit multiple sites in a single experiment.
    CRISPR does not require the presence of a PAM sequence to be adjacent to the target sequence.
    None of the above

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: CRISPR can edit multiple sites in a single experiment.
