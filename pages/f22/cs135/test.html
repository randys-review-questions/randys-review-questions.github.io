1. Which of the following is true regarding the relationship between Artificial Intelligence and Machine Learning?

    Artificial Intelligence is a subfield of Machine Learning.
    Machine Learning is a subfield of Artificial Intelligence.
    Artificial Intelligence and Machine Learning are equivalent fields.
    Artificial Intelligence and Machine Learning are completely unrelated from each other.

Correct!

Incorrect. The correct answer was: Machine Learning is a subfield of Artificial Intelligence.

2. When using gradient descent to update weights, the change in weight is...

    in the same direction as the loss gradient.
    in the opposite direction as the loss gradient.
    in the same direction as the loss gradient if the loss gradient is positive and in the opposite direction if the loss gradient is negative
    in the same direction as the loss gradient if the loss gradient is negative and in the opposite direction if the loss gradient is positive.

Correct!

Incorrect. The correct answer was: in the same direction as the loss gradient if the loss gradient is negative and in the opposite direction if the loss gradient is positive.

3. Suppose that you are using polynomial regression to solve a supervised learning problem. As the degree of the polynomial increases, the training error will...

    increase.
    decrease.
    increase then decrease.
    decrease then increase.

Correct!

Incorrect. The correct answer was: decrease.

4. When completing a certain round of 20-fold cross-validation, what proportion of the data is used to train the model?

    5%
    20%
    80%
    95%

Correct!

Incorrect. The correct answer was: 95%

5. All of the following are classifiers EXCEPT...

    Ridge Regression
    Logistic Regression
    Multi-Layer Perceptron
    Decision Trees

Correct!

Incorrect. The correct answer was: Ridge Regression

6. When compared to One-Versus-One classification, in One-Versus-All Classification, meeting the requirements for linear separability is _____, and _____ classifiers are necessary.

    easier, fewer
    easier, more
    harder, fewer
    harder, more

Correct!

Incorrect. The correct answer was: harder, fewer

7. Consider the following statistics:

True Positives10
True Negatives15
False Positives20
False Negatives30

What is the true positive rate?

    25%
    33.33%
    40%
    50%

Correct!

Incorrect. The correct answer was: 25%

8. Suppose that in your training data set, you had the following three data points (format: \([x_1, x_2]\)): \([0, 100]\), \([5, 50]\), and \([10, 0]\), and you wanted to use feature standardization for feature engineering. What would be the data points after feature standardization?

The feature standard deviations are: \([\sigma_1, \sigma_2] = [5, 50]\)

    \([0, 1]\), \([0.5, 0.5]\), and \([1, 0]\)
    \([0, 1]\), \([0.05, 0.5]\), and \([0.1, 0]\)
    \([-1, 1]\), \([0, 0]\), and \([1, -1]\)
    \([0, 2]\), \([1, 1]\), and \([2, 0]\)

Correct!

Incorrect. The correct answer was: \([-1, 1]\), \([0, 0]\), and \([1, -1]\)

9. The "kernel trick" can allow for the learning of a separator for data that is not linearly separable. How?

    New features are added that are linear combinations of existing features.
    New features are added that are non-linear combinations of existing features.
    A kernel function transforms the data into a space where the points are linearly separable.
    The ability to learn weights across multiple hidden layers allows for a non-linear separator.

Correct!

Incorrect. The correct answer was: A kernel function transforms the data into a space where the points are linearly separable.

10. In a Feed-Forward neural network, which of the following processes can be parallelized?

    Learning the weights of different neurons within the same layer
    Learning the weights of different neurons in different layers that are connected to each other
    All of the above
    None of the above

Correct!

Incorrect. The correct answer was: Learning the weights of different neurons within the same layer

11. Suppose that you have a 4-sided die that is rigged so that it lands on 1 50% of the time, lands on 2 25% of the time, and lands on 3 and 4 12.5% of the time each. What is the number of bits of information yielded while rolling the die?

This is the Shannon Entropy with a log base of 2.

    0.875
    1.75
    2
    2.25

Correct!

Incorrect. The correct answer was: 1.75

12. Reinforcement learning problems are...

    regression supervised learning problems.
    classification supervised learning problems.
    semi-supervised learning problems.
    unsupervised learning problems.

Correct!

Incorrect. The correct answer was: semi-supervised learning problems.

13. In a Markov decision process, if the time horizon (\(T\)) is infinite, we should make sure that the discount rate (\(\gamma\)) is...

    positive.
    negative.
    less than 1.
    greater than 1.

Correct!

Incorrect. The correct answer was: less than 1.

14. For a K Nearest Neighbors classifier, looking at a large number of neighbors is likely to result in...

    an underfit model.
    an appropriately fit model.
    an overfit model.
    Number of neighbors has no effect on underfitting/overfitting.

Correct!

Incorrect. The correct answer was: an underfit model.

15. The use of inverse document frequency in a "Bag of Words" document model seeks to emphasize...

    words found a few times in the average document over words found many times in the average document.
    words found many times in the average document over words found a few times in the average document.
    words found in a few documents over words found in many documents.
    words found in many documents over words found in a few documents.

Correct!

Incorrect. The correct answer was: words found in a few documents over words found in many documents.
