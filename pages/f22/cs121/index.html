Randy's Review Questions

CS 121: Software Engineering

Based on content taught by Jeffrey Foster in Fall 2022. All programming questions use the Java language.

1. Which of the following statements about Java is incorrect?

    Omitting a break statement at the end of a case in a switch statement will lead to code "falling through" to the next case.
    Classes are used as namespaces in Java programs.
    A class that implements an interface may only define methods that are declared in the interface.
    Strings are immutable.

Correct!

Incorrect. The correct answer was: A class that implements an interface may only define methods that are declared in the interface.

2. Suppose that the Ryan class inherits from the Person class, and they both contain a speak method. The speak method in the Person class involves a call to print Hello! The speak method in the Ryan class involves a call to print My name is Ryan! Consider the following lines of code (assume it's wrapped in some main() method). What is printed?

Person x = new Ryan();
x.speak();
    Hello!
    My name is Ryan!
    Hello! followed by My name is Ryan!
    This code would not compile.

Correct!

Incorrect. The correct answer was: My name is Ryan!

3. In "On the Criteria to be Used in Decomposing Systems into Modules" by D. L. Parnas, what is the main "criterion" that he describes should be used?

    Steps of the Algorithm
    Information Hiding
    Team Member Strengths and Weaknesses
    Using one module for everything

Correct!

Incorrect. The correct answer was: Information Hiding

4. All of the following are true statements about iterators in Java EXCEPT...

    The use of external iterators exposes details about data structure implementation.
    Using iterators for a data structure uses more heap memory than traversing a data structure directly.
    When an iterator and data structure implement the necessary methods, the following syntactic sugar can be used: for (Object elem : elems) {}
    A data structure traversed by an Iterator should implement the Iterable interface.

Correct!

Incorrect. The correct answer was: The use of external iterators exposes details about data structure implementation.

5. What design pattern uses the functionality of one implementation of an interface to support a different interface?

    Decorator
    Proxy
    Adapter
    Singleton

Correct!

Incorrect. The correct answer was: Adapter

6. Which of the following design patterns utilizes double dispatch?

    Iterator
    Delegation
    Observer
    Visitor

Correct!

Incorrect. The correct answer was: Visitor

7. Suppose that m is a Method object representing an instance method. Typically, that method would be invoked with m.invoke(Object receiver, Object [] args). If m instead represented a class method, what would be passed in as the receiver?

    A new instance of the class
    The class itself
    null
    Any of the above will suffice.

Correct!

Incorrect. The correct answer was: Any of the above will suffice.

8. What kind of testing checks that the components of an entire system are working together correctly?

    Unit Test
    Regression Test
    Integration Test
    Acceptance Test

Correct!

Incorrect. The correct answer was: Integration Test

9. What is refactoring?

    Fixing old bugs that reappear after making changes to software
    Changing software architecture without changing behavior to support new features
    Incorporating new features into software
    The conversion of hardware to its software equivalent

Correct!

Incorrect. The correct answer was: Changing software architecture without changing behavior to support new features

10. In the Model-View-Controller architecture, which of the following acts as a database?

    Model
    View
    Controller
    Server

Correct!

Incorrect. The correct answer was: Model

11. In layered architecture, each layer _____ the layer below it, and changing one layer affects at most __ other(s).

    is a client of, 1
    is a client of, 2
    provides services to, 1
    provides services to, 2

Correct!

Incorrect. The correct answer was: is a client of, 2

12. In Java, we can replicate the behavior of a ReentrantLock using...

    a bool.
    an int, but not with a bool.
    a List, but not with an int.
    It is impossible to replicate ReentrantLock behavior in Java.

Correct!

Incorrect. The correct answer was: It is impossible to replicate ReentrantLock behavior in Java.

13. In the absence of locks, what does it mean for a procedure to be "atomic"?

    The procedure can be used as a key in a HashMap.
    The procedure involves only one line of Java code.
    The procedure involves only one line of Java code AND does not call any library functions.
    The procedure involves only one machine instruction.

Correct!

Incorrect. The correct answer was: The procedure involves only one machine instruction.

14. What debugging approach involves executing a binary search on some input to find a smaller input that exhibits a bug?

    Alpha Debugging
    Beta Debugging
    Gamma Debugging
    Delta Debugging

Correct!

Incorrect. The correct answer was: Delta Debugging

15. A system with _____ ensures that no unauthorized individual can change stored data.

    integrity
    availability
    confidentiality
    cross-site scripting

Correct!

Incorrect. The correct answer was: integrity
