Randy's Review Questions

TAing CS15: Data Structures

Based on the experience of TAing CS15 in Fall 2021 under Milod Kazerounian. 70% of this quiz is about course content, whereas the remaining 30% is about the TA experience specifically.

1. For each assignment, a grader should spend a maximum of _____ debugging cascading errors.

    10-15 minutes
    45-60 minutes
    2 hours
    16 hours

Correct!

Incorrect. The correct answer was: 10-15 minutes

2. If a student's submission does not compile, what command would give an indication of why?

Let's assume that the autograder already ran and failed prior to typing this command.

    ./run_autograder --diff
    ./run_autograder --filter memerror
    ./run_autograder --compileLog
    There is no way to debug compiler errors.

Correct!

Incorrect. The correct answer was: ./run_autograder --compileLog

3. When using HitMe (the site that organizes who grades what), what happens if a TA gets someone they know?

    The TA should grade them more generously in order to not ruin their friendship.
    The TA should grade them less generously to try to offset any unconscious bias.
    The TA should try to grade them with as little bias as possible.
    The TA should arrange to trade assignments with another TA.

Correct!

Incorrect. The correct answer was: The TA should arrange to trade assignments with another TA.

4. In which of the following operations does a LinkedList have a better runtime than ArrayLists?

    Insertion at the front
    Removing from the back
    Binary Search
    Linear Search

Correct!

Incorrect. The correct answer was: Insertion at the front

5. What is the height of a leaf of an AVL tree?

    \(-1\)
    \(0\)
    \(1\)
    \(\log_2 (num\_nodes)\)

Correct!

Incorrect. The correct answer was: \(0\)

6. You implement a hash table using linear probing in C++. Each element is a struct containing the data being stored (key and value) and a boolean "filled" flag. What's missing?

    Nothing!
    Pointer to start of array
    Hash value
    Boolean "removed" flag

Correct!

Incorrect. The correct answer was: Boolean "removed" flag

7. Which data structure best models task interruption and resumption?

    Queue
    Stack
    Binary Search Tree
    Graph

Correct!

Incorrect. The correct answer was: Stack

8. What is the worst case runtime for inserting an element in a binary search tree?

    \(O(1)\)
    \(O(\log n)\)
    \(O(\sqrt{n})\)
    \(O(n)\)

Correct!

Incorrect. The correct answer was: \(O(n)\)

9. Which sorting algorithm is ideal for when all elements are within a bounded range (and the range size is small)?

    Bubble sort
    Bucket/counting sort
    Merge sort
    Intelligent design sort

Correct!

Incorrect. The correct answer was: Bucket/counting sort

10. What is the runtime of breadth first search and depth first search for a graph represented as an adjacency matrix?

    \(O(|V|)^2\)
    \(O(|E|)^2\)
    \(O(|V|+|E|)\)
    \(O(|V|*|E|)\)

Correct!

Incorrect. The correct answer was: \(O(|V|)^2\)
