1. What is the antiderivative of \(\frac{1}{x}\), where \(x>0\)?

    \(\infty\)
    \(- \infty\)
    \(- \frac{1}{x^2}\)
    \( \ln{(x)} \)
    \( ln{(x)} + C \)

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: \( ln{(x)} + C \)

2. Based on the Mean Value Theorem, if it took your car 2 hours to travel 120 miles, then your speedometer must have read _____ at at least one point in the journey.

    0 mph
    40 mph
    60 mph
    80 mph
    120 mph

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: 60 mph

3. A function is considered a continuous function if...

    it is continuous at every point from \(- \infty\) to \(+ \infty\)
    it is continuous at every point from \(- \infty\) to \(0\)
    it is continuous at every point from \(0\) to \(+ \infty\)
    it is continuous at every point of its domain
    it is continuous at every discontinuity

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: it is continuous at every point of its domain

4. Which of the following are derived correctly? Select two options.

    \(f(x) = x^3+2x^2+4\); \(f'(x) = \frac{1}{4}x^4+\frac{2}{3}x^3+4x+10\)
    \(g(x) = 1874\); \(g'(x) = 0\)
    \(h(x) = \cos{(x)}\); \(h'(x) = \sin{(x)}\)
    \(j(x) = \sin^2{(x)}; j'(x) = 2 \sin{(x)} \cos{(x)}\)

Correct!

Incorrect. Try again.

Incorrect. The correct answers were: \(g(x) = 1874\); \(g'(x) = 0\), \(j(x) = \sin^2{(x)}; j'(x) = 2 \sin{(x)} \cos{(x)}\)

5. Which of the following is a method of finding distance?

    Taking the integral of velocity
    Taking the derivative of position
    Taking the integral of acceleration
    Taking the integral of the absolute value of velocity
    Taking the integral of the absolute value of acceleration

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Taking the integral of the absolute value of velocity

6. Which of the following is/are (a) true statement(s)? Select all that apply.

    If \(f(x)\) is increasing, then LRAM will be an overestimation of the area underneath the curve
    If \(f(x)\) is decreasing, then RRAM will be an underestimation of the area underneath the curve
    If \(f(x)\) is concave up, then the trapezoidal approximation is an overestimation of the area underneath the curve
    If \(f(x)\) is concave down, then the trapezoidal approximation is an underestimation of the area underneath the curve

Correct!

Incorrect. Try again.

Incorrect. The correct answers were: If \(f(x)\) is decreasing, then RRAM will be an underestimation of the area underneath the curve, If \(f(x)\) is concave up, then the trapezoidal approximation is an overestimation of the area underneath the curve, If \(f(x)\) is concave down, then the trapezoidal approximation is an underestimation of the area underneath the curve

9. The ratio test involves finding the limit (\(L\)) as \(n\) approaches infinity of the absolute value of the ratio between the "\(n\)th plus one" term of a sequence and the \(n\)th term. What can be concluded from this test?

    If \(L<0\), the series converges; if \(L=0\), the series converges conditionally; if \(L>0\), the series diverges
    If \(L<1\), the series converges; if \(L=1\), the series converges conditionally; if \(L>1\), the series diverges
    If \(L=0\), the series converges; if \(L=\infty\) or \(L=- \infty\), the series diverges; otherwise, the test is inconclusive
    If \(L=0\), the series diverges; if \(L=\infty\) or \(L=- \infty\), the series converges; otherwise, the test is inconclusive
    If \(L<1\), the series converges; if \(L>1\), the series diverges; if \(L=1\), the test is inconclusive

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: If \(L<1\), the series converges; if \(L>1\), the series diverges; if \(L=1\), the test is inconclusive

10. Which of the following is an example of a logistic differential equation?

    \(\frac{dy}{dt} = ky(M-y)\)
    \(\frac{dy}{dt} = ky\)
    \(y = Ae^{rt}\)
    \(y = \frac{M}{1+Ae^{-Mkt}}\)
    \(\frac{dy}{dt} = kt(M-t)\)

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: \(\frac{dy}{dt} = ky(M-y)\)
