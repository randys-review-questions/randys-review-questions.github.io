1. When solving a Sudoku puzzle, each number from 1 to 9 can only appear once in each...

    row.
    column.
    3x3 grid.
    All of the above

Correct!

Incorrect. The correct answer was: All of the above

2. Which kind of machine learning is useful in finding unknown patterns in data?

    Supervised Learning
    Unsupervised Learning
    Reinforcement Learning
    Punishment Learning

Correct!

Incorrect. The correct answer was: Unsupervised Learning

3. An actor/celebrity's Bacon number is...

    the number of links between that actor/celebrity and Kevin Bacon.
    the number of people that intercept the links between the actor/celebrity and Kevin Bacon.
    the number of times that the actor/celebrity has interacted with Kevin Bacon in their lifetime.
    the number of slices of bacon that the actor/celebrity ate that day.

Correct!

Incorrect. The correct answer was: the number of links between that actor/celebrity and Kevin Bacon.

4. The controversy surrounding regular surveillance stems from the trade-off between...

    group cohesiveness and problem-solving effectiveness.
    challenging tests and stress-free tests.
    people's privacy and security.
    providing objective proof and manipulation of evidence.

Correct!

Incorrect. The correct answer was: people's privacy and security.

5. Which of the following should NOT be considered when designing facial recognition software?

    Eye Placement
    Bone Structure
    Distance between nose and ears
    Glasses

Correct!

Incorrect. The correct answer was: Glasses
