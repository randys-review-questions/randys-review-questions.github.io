1. Which of the following statements regarding working on a team is incorrect?

    Any issues or complaints regarding a team member's conduct should be addressed sooner rather than later.
    Stronger teams result from having more intelligent team members.
    It is important for each team member to contribute in the same proportion as other team members.
    Dictatorships lead to better results than democracies.

Correct!

Incorrect. The correct answer was: Stronger teams result from having more intelligent team members.

2. Which of the following is a necessary characteristic of a compiler?

"Source program" refers to the program to compile.

    It must statically analyze the source program.
    It must transform the source program into machine code.
    It must execute the source program.
    All of the above

Correct!

Incorrect. The correct answer was: It must statically analyze the source program.

3. Consider the following OCaml expression. What is returned?

let foo xs =
    let bar x y = x <> y in
    List.fold_left bar true xs
in
foo [true; false; true]
    true
    false
    [true; false; true]
    [false; true; false]

Correct!

Incorrect. The correct answer was: true

4. Consider the following OCaml declaration. What is the type of foo?

let foo p = match p with
 | (None, None) -> None
 | (None, Some(y)) -> Some(y)
 | (Some(x), None) -> Some(x)
 | (Some(x), Some(y)) -> Some(x + y)
    ('a * 'a) option -> int option
    'a option * 'a option -> int option
    (int * int) option -> int option
    int option * int option -> int option

Correct!

Incorrect. The correct answer was: int option * int option -> int option

5. The scanning step of a compiler takes in _____ as input and produces _____.

    a stream of characters, a stream of tokens
    a stream of characters, an abstract syntax tree
    a stream of tokens, an abstract syntax tree
    an abstract syntax tree, the target program

Correct!

Incorrect. The correct answer was: a stream of characters, a stream of tokens

6. Which of the following NFAs (nondeterminstic finite automata) correctly accept all strings in the language defined by the regular expression \((012)*\) and reject all other strings?

    a
    b
    c
    d

Correct!

Incorrect. The correct answer was: c

7. Consider the following grammar for a simple example language that works with alphanumeric strings and the following partial implementation of parser.mly (input to the Menhir parser generator). In this language, adding string \(b\) to string \(a\) with the + operator concatenates \(b\) to the end of \(a\), and subtracting string \(b\) from string \(a\) with the - operator removes all the characters in \(b\) from \(a\). For example, the expression abc + 123 evalutes to abc123, and the expression abc123 - 2b24 evalutes to ac13. Assume that all whitespace is ignored, the + symbol lexes to the PLUS token, the - symbol lexes to the MINUS token, and any sequence of alphanumeric characters lexes to the STRING token.

What does abc123 - 123 + cde345 - 345 evaluate to?

// Grammar:
expr:
    expr PLUS expr
  | expr MINUS expr
  | STRING

// Partial implementation of parser.mly:
%token PLUS MINUS
%token <string> STRING

%right MINUS
%left PLUS
    ab
    ab3
    abc
    abccde

Correct!

Incorrect. The correct answer was: ab3

8. Consider the following grammar:

\(a \to b + b\)

\(b \to \text{NUMBER} * \text{NUMBER}\)

\(b \to \text{NUMBER}\)

Which of these options correctly underlines the handle of the following right sentential form?

\(\text{NUMBER} + \text{NUMBER} * \text{NUMBER}\)

    \(\underline{\text{NUMBER}} + \text{NUMBER} * \text{NUMBER}\)
    \(\text{NUMBER} + \text{NUMBER} * \underline{\text{NUMBER}}\)
    \(\underline{\text{NUMBER} + \text{NUMBER}} * \text{NUMBER}\)
    \(\text{NUMBER} + \underline{\text{NUMBER} * \text{NUMBER}}\)

Correct!

Incorrect. The correct answer was: \(\text{NUMBER} + \underline{\text{NUMBER} * \text{NUMBER}}\)

9. Symbol tables that map user-defined names to information about the "meaning" of such names (e.g. their types) are useful in the implementation of which compiler "pass"?

    Lexing
    Parsing
    Semantic Analysis
    Code Generation

Correct!

Incorrect. The correct answer was: Semantic Analysis

10. Consider the following lines of 3-Address Code (not in SSA form). On what line does the basic block starting at label L2 end?

Note: the instruction bz L, a means: jump to label L if the value stored in a is true.

1  L1: a = add 2, 3
2  L2: b = add 5, a
3      c = shl b, 3
4  L3: a = sub a, 1
5      bz L2, a
6      jmp L4
7      c = shr c, 3
8  L4: ret c
    3
    5
    6
    7

Correct!

Incorrect. The correct answer was: 5

11. When using the OCaml API for LLVM, which object maintains global data?

    The builder
    The context
    The element
    The module

Correct!

Incorrect. The correct answer was: The context

12. The reference counting algorithm for garbage collection works in all of the following scenarios EXCEPT...

    when there exists a heap object that references another heap object not referenced on the stack or in a register.
    when there exists a heap object that references another heap object which references the first heap object.
    when there exists a heap object referenced on the stack but not referenced in a register.
    All of the above are scenarios where the reference counting algorithm works.

Correct!

Incorrect. The correct answer was: when there exists a heap object that references another heap object which references the first heap object.

13. An object's virtual table (used in the implementation of inheritance) contains which of the following?

    Pointers to its methods
    Its data members
    Pointers to the data members and methods of its superclass
    Both the first two options and a pointer to its superclass's virtual table

Correct!

Incorrect. The correct answer was: Pointers to its methods

14. Which of the following techniques is not a compiler optimization on its own, but can lead to further optimizations when paired with other techniques?

    Constant Folding
    Common Sub-Expression Elimination
    Copy Propagation
    Dead Code Removal

Correct!

Incorrect. The correct answer was: Copy Propagation

15. Consider the following lines of 3-Address Code. Which variables are live between the instruction on line 3 and the instruction on line 4?

1  a = add 2, 3
2  c = sub a, 1
3  b = sub a, 2
4  a = add b, 3
5  b = sub a, 1
6  c = add a, 1
    b only
    a and b only
    a and c only
    a, b, and c

Correct!

Incorrect. The correct answer was: b only
