1. Consider the following lines of R. Which of these options has the same effect?

f <- function() {
    a <- 1
    b <- c(2, 3)
    x <- c(a, b)
    x
}
    f <- function() { c(1, 2, 3) }
    f <- function() { c(1, c(2, 3)) }
    f <- function() { list(1, c(2, 3)) }
    Both the first option and the second option.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Both the first option and the second option.

3. Let \(p\) equal the success probability per trial in a binomial distribution and let \(s\) equal the number of trials. This distribution closely approximates a poisson distribution with \(\lambda = \) _____ when \(p\) is _____ and \(s\) is high.

    \(p * s\), low
    \(p * s\), high
    \(\frac{p}{s}\), low
    \(\frac{p}{s}\), high

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: \(p * s\), low

4. Suppose that scores is a 25 x 5 matrix of test scores such that scores[i, j] is the score that student i received for test j. Further suppose that toPct is a function that takes in a single numeric value and returns that value as a percentage (i.e. multiplies the inputted value by 100). Which of the following stores, in scoresPct, a matrix equivalent to scores except that all test scores are in percentage form?

    scoresPct <- apply(scores, 1, toPct)
    scoresPct <- apply(scores, 2, toPct)
    scoresPct <- apply(scores, 3, toPct)
    scoresPct <- apply(scores, c(1, 2), toPct)

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: scoresPct <- apply(scores, c(1, 2), toPct)

8. Count data is often zero-inflated due to not everything being measured. Such zero-inflated data can be modelled with...

    a Gamma-Poisson distribution.
    a mixture of Dirichlet priors.
    a mixture model consisting of a model for the zero counts and another model for the nonzero counts.
    a mixture model consisting of a model for some of the zero counts and another model for the remaining counts.

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: a mixture model consisting of a model for some of the zero counts and another model for the remaining counts.

9. You associate Carm ratings with an index of \(0\) and Dewick ratings with an index of \(1\), and you fit those ratings to a linear model \(y = \beta_0 + \beta_1 x\), which is used to perform a t-test. Which of the following would be the mean rating of Dewick?

    \(\beta_0\)
    \(\beta_1\)
    \(\beta_0 + \beta_1\)
    \(\frac{\beta_0}{\beta_1}\)

10. Which of the following returns the cell types that have an average of more than 1000 mitochondria per cell?

Assume that the dplyr library has already been included.

    filter(select(cells, num_mito > 1000), type)

12. In R, lm(y ~ x) fits the data in y and x to the model \(y = \beta_0 + \beta_1 x\). Which of the following lines of code would instead fit the data to the model \(y = \beta_1 x\)?

    lm(y ~ x - 1)
    lm(y ~ x + 1)
    lm(y ~ -x)
    lm(y ~ 2x)

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: lm(y ~ x - 1)
