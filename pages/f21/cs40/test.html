1. Which of the following pertaining to unboxed/boxed data structures is incorrect?

    In a boxed data structure, the user needs to specify the size of each element upon initialization.
    Accessing elements in an unboxed data structure results in better spatial locality.
    A boxed data structure holds void pointers.
    In an unboxed structure, the implementation handles deallocating elements.

Correct!

Incorrect. The correct answer was: In a boxed data structure, the user needs to specify the size of each element upon initialization.

2. A "magic number" of P5 at the top of an image file indicates that it is a(n)...

    plain portable bitmap.
    raw portable bitmap.
    plain portable graymap.
    raw portable graymap.

Correct!

Incorrect. The correct answer was: raw portable graymap.

3. Consider the following lines of C code. What is printed?

int x = 256;
int *int_p = &x;
void *void_p = int_p;
char *char_p = void_p;
printf("%c\n", *char_p);
    0
    256
    The null character
    This code does not compile.

Correct!

Incorrect. The correct answer was: The null character

4. The apply function inputted into Hanson's Table_map function must take in a void * for the key, a void ** for the value, and a void * for the closure. What is the purpose of passing in a double pointer for the value?

    This allows an apply function to change the value stored in the table, which is a void *.
    The Hanson table by default stores void * for keys and void ** for values.
    The key and value inputted into the function cannot be the same type, so passing in a void ** helps get around that.
    The use of a double pointer allows for a polymorphic table structure.

Correct!

Incorrect. The correct answer was: This allows an apply function to change the value stored in the table, which is a void *.

9. A direct mapped cache is the same as...

10. Consider the following lines of code. Suppose that you have a fully associative cache with a blocksize of 2 bytes, and this cache has 4 lines total. What is the hit rate (assume that i and j are stored in registers)?

    for (int j = 0; j < 6; j++) {

    25%
    33.33%
    50%

Incorrect. The correct answer was: 0%
