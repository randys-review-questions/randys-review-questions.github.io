1. In Night by Elie Wiesel, the protagonist is named...

    Elie
    Sam
    Amir
    Lucentio

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Elie

    Snow

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Snow

Incorrect. Try again.

Incorrect. The correct answer was: iHop
