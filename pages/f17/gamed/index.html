Randy's Review Questions

Game Design and Development

Based on content taught by Dan Warburton in Spring 2018.

1. All of the following are necessary to a game EXCEPT...

    A story
    High quality graphics
    A problem
    A solution

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: High quality graphics

2. Games should be...

    Impossible to beat
    Beatable, but challenging
    Very easily beatable
    Completely blank

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Beatable, but challenging

3. In blender, when designing, one should use...

    Notation mode
    Perspective mode
    Orthographic mode
    Orthopedic mode

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Orthographic mode

4. In blender, edit mode allows you to...

    Edit objects as a whole
    Edit specific properties of one object
    Edit specific properties of multiple objects
    Undo and redo actions

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Edit specific properties of one object

5. In blender, which modifier would generally make the surface of an object more smooth?

    Distort
    Mirror
    Subdivision Surface
    Particle Edit

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Subdivision Surface

6. In an animation in blender, a cube travels 2 units on the x axis, then 2 units on the z axis. You would insert a _____ between those two movements.

    Yellow Frame
    Green Frame
    Keyframe
    Transition

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Keyframe

7. The first step of the Engineering and Design Process is...

    Define the Problem
    Brainstorm
    Research
    Prototype

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Define the Problem

8. Throughout the engineering and design process, one should always be...

    Brainstorming
    Researching
    Writing
    Evaluating

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Evaluating

9. An attribute in stencyl is basically a(n)...

    Variable
    Code
    Actor
    Sound

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Variable

10. In Stencyl, the default animation on an actor is the one with a...

    Circle
    Square
    Star
    Triangle

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Star
