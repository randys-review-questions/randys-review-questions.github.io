Randy's Review Questions

Algebra II

Based on content taught by Michael Lougee in the 2017-2018 school year.

1. When graphing a system of inequalities, when a line is not included, it is drawn as...

    A ray
    A line segment
    A solid line
    A dotted line

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: A dotted line

2. In a function...

    Each input is paired with one output
    Each output is paired with one input
    Each input is paired with exactly one output
    Each output is paired with exactly one input

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Each input is paired with exactly one output

3. Factor: \(x^2-y^2\)

    \((x-y)^2\)
    \((x+y)^2\)
    \((x-y)(x+y)\)
    \(x^2+xy+y^2\)

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: \((x-y)(x+y)\)

4. The vertex of a quadratic function can be determined from which equation?

    \(x = - \frac{b}{a}\)
    \(x = - \frac{b}{2a}\)
    \(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)
    APUSH

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: \(x = - \frac{b}{2a}\)
