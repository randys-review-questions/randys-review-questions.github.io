1. It is impossible for a system of linear equations to have _____ solutions.

    0
    1
    2
    an infinite number of

Correct!

Incorrect. The correct answer was: 2

2. Let \(\textbf{v}_1 = (1, 0, 0)\), \(\textbf{v}_2 = (0, 1, 0)\), and \(\textbf{v}_3 = (7, 3, 0)\). All vectors in \(Span\{\textbf{v}_1, \textbf{v}_2, \textbf{v}_3\}\) make up...

    a point in \(\mathbb{R}^3\)
    a line in \(\mathbb{R}^3\)
    a plane in \(\mathbb{R}^3\)
    the entire \(\mathbb{R}^3\)

Correct!

Incorrect. The correct answer was: a plane in \(\mathbb{R}^3\)

3. Let \(A\) be an \(m \times n\) matrix. What is a necessary condition for the columns of \(A\) to span \(\mathbb{R}^m\)?

    \(m \leq n\)
    \(m \geq n\)
    \(m = n\)
    \(m \neq n\)

Correct!

Incorrect. The correct answer was: \(m \leq n\)

4. A transformation from \(\mathbb{R}^n\) to \(\mathbb{R}^m\) is linear iff for any \(u, v\) in \(\mathbb{R}^n\) and any scalar \(c\)...

    \(T(u+v) = T(u)+T(v)\)
    \(T(cu) = cT(u)\)
    All of the above
    None of the above

Correct!

Incorrect. The correct answer was: All of the above

5. Let \(T\) be a linear transformation from \(\mathbb{R}^n\) to \(\mathbb{R}^m\) and let \(A\) be the standard matrix for \(T\). \(T\) is one-to-one iff...

    the columns of \(A\) span \(\mathbb{R}^m\).
    the columns of \(A\) do not span \(\mathbb{R}^m\).
    the columns of \(A\) are linearly independent.
    the columns of \(A\) are linearly dependent.

Correct!

Incorrect. The correct answer was: the columns of \(A\) are linearly independent.

6. All of the following are always true for matrices \(A\), \(B\), and \(C\) EXCEPT...

Assume that \(A\), \(B\), and \(C\) have sizes where all operations in the answer options are defined.

    \(AB = BA\)
    \(A(BC) = (AB)C\)
    \(A(B+C) = AB+AC\)
    \((B+C)A = BA+CA\)

Correct!

Incorrect. The correct answer was: \(AB = BA\)

7. Let \(A\) be an \(m \times n\) matrix. All of the following are logically equivalent to "\(A\) is invertible" EXCEPT...

    \(A\) is row equivalent to the identity matrix.
    \(nullity(A) = 0.\)
    the columns of \(A\) are linearly dependent.
    None of these are logically equivalent to "\(A\) is invertible."

Correct!

Incorrect. The correct answer was: None of these are logically equivalent to "\(A\) is invertible."

8. Let \(A\) be a \(3 \times 3\) matrix. Suppose that \(det(A) = 5\). What is \(det(2A)\)?

    5
    10
    30
    40

Correct!

Incorrect. The correct answer was: 40

9. Let \(A\) be a matrix. What is \(Nul(A)\)?

    The set of all possible linear combinations of the columns of \(A\).
    The set of all possible linear combinations of the rows of \(A\).
    The set of all solutions to the homogeneous equation \(Ax = 0\).
    The set of all solutions to the nonhomogeneous equation \(Ax = b\).

Correct!

Incorrect. The correct answer was: The set of all solutions to the homogeneous equation \(Ax = 0\).

10. Let \(P_n\) be a vector space containing all polynomials with degree less than or equal to \(n\). Which of the following is true?

    \(P_2\) is a subspace of \(P_3\)
    \(P_3\) is a subspace of \(P_2\)
    \(\mathbb{R}^2\) is a subspace of \(\mathbb{R}^3\)
    Both the first option and the third option are true.

Correct!

Incorrect. The correct answer was: \(P_2\) is a subspace of \(P_3\)

11. Let \(A\) be an \(n \times n\) matrix and let \(2\) be an eigenvalue of \(A\). What is the set of all eigenvectors corresponding to this eigenvalue?

    \(Nul(A)\)
    \(Nul(A-2)\)
    \(Nul(A-2I)\)
    \(Nul(A-2I) - \{0\}\)

Correct!

Incorrect. The correct answer was: \(Nul(A-2I) - \{0\}\)

12. What does it mean for a matrix \(A\) to be diagonalizable?

    \(A\) is invertible, and the inverse of \(A\) is also invertible.
    \(A\) is row equivalent to a diagonal matrix.
    \(A\) is similar to a diagonal matrix.
    \(A\) is a triangular matrix.

Correct!

Incorrect. The correct answer was: \(A\) is similar to a diagonal matrix.

13. Two vectors are orthogonal iff...

    their inner product is 0.
    their inner product is 1.
    their cross product is 0.
    their cross product is 1.

Correct!

Incorrect. The correct answer was: their inner product is 0.

14. What is the orthogonal complement of \(Nul(A)\)?

    \(Nul(A^T)\)
    \(Nul(A^{-1})\)
    \(Row(A)\)
    \(Col(A)\)

Correct!

Incorrect. The correct answer was: \(Row(A)\)

15. The three lines in the diagram below correspond to three linear equations on a 2-D plane. Which point represents a least squares solution of the system of those three linear equations?

    a
    b
    c
    d

Correct!

Incorrect. The correct answer was: d
