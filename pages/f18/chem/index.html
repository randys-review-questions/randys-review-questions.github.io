Randy's Review Questions

AP Chemistry

Based on content that I learned myself using resources provided by Stephanie Bertenshaw in the 2018-2019 school year.

1. What two conditions would make a reaction thermodynamically favorable?

    An increase in enthalpy
    A decrease in enthalpy
    An increase in entropy
    A decrease in entropy

Correct!

Incorrect. Try again.

Incorrect. The correct answers were: A decrease in enthalpy, An increase in entropy

2. A central atom with 5 bonds and 1 lone pair has what electron domain geometry?

    Trigonal pyramidal
    Tetrahedral
    Trigonal bipyramidal
    Square planar
    Octahedral

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Octahedral

3. The ratio of the effusion rates of two gases is inversely proportional to...

    the ratio between the molar masses
    the ratio between the square of the molar masses
    the ratio between the square root of the molar masses
    the ratio between the volumes
    the ratio between the square of the volumes

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: the ratio between the square root of the molar masses

4. In a first order reaction, which of the following relationships produces a linear graph?

If there are multiple correct answers, select the one that is conventionally used.

    [reactant] vs. time
    -[reactant] vs. time
    log[reactant] vs. time
    ln[reactant] vs. time
    1/[reactant] vs. time

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: ln[reactant] vs. time

5. A Lewis base is a base that...

    donates \(H^+\) ions in solution
    accepts \(H^+\) ions in solution
    donates \(OH^-\) ions in solution
    donates electron pairs
    accepts electron pairs

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: donates electron pairs

6. When the oxidation number of an element decreases in a redox reaction, that element is...

    Oxidized
    Reduced
    Heated
    Cooled
    Laughed At

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: Reduced

7. The pH at the equivalence point of a strong acid-strong base titration should be...

    less than 7.00 only at \(25\) \(^{\circ} C\)
    at 7.00 only at \(25\) \(^{\circ} C\)
    greater than 7.00 only at \(25\) \(^{\circ} C\)
    at 7.00 in all conditions
    not equal to 7.00 in all conditions

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: at 7.00 only at \(25\) \(^{\circ} C\)

8. The van't Hoff factor (\(i\)) in the context of colligative properties in solutions is...

    the number of atoms per formula unit
    the number of particles per formula unit
    the number of ions per formula unit
    the number of molecules per formula unit
    the number of moles of atoms per formula unit

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: the number of particles per formula unit

9. Which orbital, when considering areas with a 95% probability density or higher, has two lobes and one doughnut shape?

    \(s\) orbital
    \(p\) orbital (\(p_x\), \(p_y\), or \(p_z\))
    \(d\) orbital (\(d_{xy}\), \(d_{yz}\), or \(d_{xz}\))
    \(d\) orbital (\(d_{x^2-y^2}\))
    \(d\) orbital (\(d_{z^2}\))

Correct!

Incorrect. Try again.

Incorrect. The correct answer was: \(d\) orbital (\(d_{z^2}\))

10. Which of the following organic structures has five carbon atoms? Select all that apply.

    Pentane
    Propane
    2-methylbutane
    2,3-dimethylpentane

Correct!

Incorrect. Try again.

Incorrect. The correct answers were: Pentane, 2-methylbutane
