Randy's Review Questions

AP Computer Science Principles

Based on content taught by John Redford in the 2018-2019 school year. All programming questions use the JavaScript language.

1. What is the minimum number of bits required to represent a letter of the alphabet?

    2
    3
    4
    5
    6

Correct!

Incorrect. The correct answer was: 5

2. Which of the following are protocols involved in the process of sending packets over the internet? Select two options.

    TCP
    TLS
    IP
    HTML

Correct!

Incorrect. The correct answers were: TCP, IP

3. What is one disadvantage to lossy compression?

    The quality of the image/sound file/video is noticeably lower
    It does not save that much space
    The original file cannot be completely recovered
    It has a low reliability
    It deletes the file's IP address

Correct!

Incorrect. The correct answer was: The original file cannot be completely recovered

4. All of the following are forms of abstraction EXCEPT...

    Programming Languages
    Binary Code
    Functions
    Parameters
    Hexadecimal Numbers

Correct!

Incorrect. The correct answer was: Binary Code

5. How many possible ways are there to encrypt a message using the Caesar Cipher?

    13
    25
    26
    27
    52

Correct!

Incorrect. The correct answer was: 25

6. A AND B = FALSE. Which of the following are possible values for A and B? Check all that apply.

    A = TRUE, B = TRUE
    A = TRUE, B = FALSE
    A = FALSE, B = TRUE
    A = FALSE, B = FALSE

Correct!

Incorrect. The correct answers were: A = TRUE, B = FALSE, A = FALSE, B = TRUE, A = FALSE, B = FALSE

7. Conditional statements are a form of...

    Iteration
    Sequencing
    Selection
    Encryption
    Decryption

Correct!

Incorrect. The correct answer was: Selection

8. Which of the following correctly displays a for loop using JavaScript?

    for (i = 0; i<3; i+) {}
    for (var i = 0, i<3, i++) {}
    for (var i = 0, i<3, i+) {}
    for (i = 0, i<3, i++) {}
    for (var i = 0; i<3; i++) {}

Correct!

Incorrect. The correct answer was: for (var i = 0; i<3; i++) {}
