Randy's Review Questions

COMP 15: Data Structures

Based on content taught by Mark Sheldon in Spring 2021. All programming questions use the C++ language.

1. Consider the following lines of code in C++. Which of the following is a stylistically better alternative?

if (isBig == true) {
    return true;
} else {
    return false;
}
    if (isBig == true) return true; else return false;
    return isBig;
    return not isBig;
    There is no way to improve the style of this code.

Correct!

Incorrect. The correct answer was: return isBig;

2. Simplify the following RPN (reverse polish notation) expression: 1 2 + 3 + 1 *

    6
    7
    16
    This is an invalid expression.

Correct!

Incorrect. The correct answer was: 6

3. What are the steps to expanding the array in the ArrayList class?

    Simply change the capacity data member
    Create another array of the same capacity and concatenate it to the end of the existing array
    Create another array of a larger size and copy over all elements
    Create a LinkedList, transfer all contents there, add elements as needed, then create another array and copy everything back.

Correct!

Incorrect. The correct answer was: Create another array of a larger size and copy over all elements

4. Consider the following line of code in C++. Which of the Big Three runs when this line is executed (prevValue is an existing instance of MyClass)?

MyClass myValue = prevValue;

    Destructor
    Copy Constructor
    Assignment Operator Overload
    None of these

Correct!

Incorrect. The correct answer was: Copy Constructor

5. Which of the following is not representative of the LinkedList implementation? Assume that no cycles result in any of these.

    Children waiting in a line; teacher keeps track of first child, and each child remembers their name and keeps track of the child behind them.
    Members of an organization; the president's phone number is kept track of, and each member keeps track of their own phone number and one other member of the organization.
    Pigs in a barn; the mother pig keeps track of one piglet, and each piglet remembers information pertaining to itself and how to find one other piglet.
    Books on a bookshelf; books are stored in designated slots on the bookshelf, and a bigger bookshelf is obtained if there's not enough room.

Correct!

Incorrect. The correct answer was: Books on a bookshelf; books are stored in designated slots on the bookshelf, and a bigger bookshelf is obtained if there's not enough room.

6. A queue is useful in all of the following situations EXCEPT...

    to model people waiting in line at a coffee shop.
    to model task interruption and resumption.
    to store the order that vertices should be checked in a breadth first traversal of a graph.
    to model order fulfillment for a restaurant.

Correct!

Incorrect. The correct answer was: to model task interruption and resumption.

7. A stack is a(n) _____ of a list.

    abstraction
    queue
    interface
    implementation

Correct!

Incorrect. The correct answer was: abstraction

8. Which of the following is NOT a representation invariant (for a correctly formed data structure)?

    An ArrayList's capacity stores the number of elements the client is storing in the array.
    The pointer to the front of an empty LinkedList is nullptr.
    For any Node N in a BinarySearchTree, all Nodes in the left subtree of N have a value less than or equal to that of N.
    All of these are correct representation invariants.

Correct!

Incorrect. The correct answer was: An ArrayList's capacity stores the number of elements the client is storing in the array.

9. What is the runtime of the following C++ lines of code?

for (int i = 0; i < n; i++) {
    for (int j = 0; j < 15; j++) {
        cout << "Boo!" << endl;
    }
}

for (int i = 0; i < n; i++) {
    cout << "Hmm..." << endl;
}
    \(O(1)\)
    \(O(n)\)
    \(O(\log n)\)
    \(O(n^2)\)

Correct!

Incorrect. The correct answer was: \(O(n)\)

10. Which of the following is incorrect about binary search?

    A binary search algorithm assumes that the list is already sorted.
    A binary search algorithm assumes that elements are comparable.
    Binary Search on a Linked List would occur in \(O(\log{n})\) time.
    All of these statements are correct about binary search.

Correct!

Incorrect. The correct answer was: Binary Search on a Linked List would occur in \(O(\log{n})\) time.

11. Which of the following is a recursive data structure?

    ArrayList
    LinkedList
    Set
    HashTable

Correct!

Incorrect. The correct answer was: LinkedList

12. What is the height of an empty tree?

    -1
    0
    1
    undefined

Correct!

Incorrect. The correct answer was: -1

13. Consider the binary tree shown below. A post order traversal would visit the Nodes in what order?

    4, 2, 6, 5, 1, 3
    4, 6, 5, 2, 1, 3
    4, 6, 5, 2, 3, 1
    6, 4, 5, 2, 3, 1

Correct!

Incorrect. The correct answer was: 4, 6, 5, 2, 3, 1

14. You insert the following elements into an AVL tree in this order: 5, 9, 7. What do you have to do to balance it?

    Left rotation
    Right rotation
    Left rotation, then a right rotation
    Right rotation, then a left rotation

Correct!

Incorrect. The correct answer was: Right rotation, then a left rotation

15. What is the best possible runtime for a comparison based sorting algorithm?

    \(O(1)\)
    \(O(n)\)
    \(O(n \log{n})\)
    \(O(n^2)\)

Correct!

Incorrect. The correct answer was: \(O(n \log{n})\)

16. When all elements are discrete values within a bounded range, and the range is small, which sorting algorithm is best?

    Insertion Sort
    Quick Sort
    Heap Sort
    Bucket/Counting Sort

Correct!

Incorrect. The correct answer was: Bucket/Counting Sort

17. Suppose that a heap is implemented using an array, and that the root Node is at index \(1\). Let \(i\) be the index representing the location of an arbitrary Node in the heap. At what index is that Node's right child (assuming that a right child exists)?

    \(\frac{i}{2}\)
    \(i+2\)
    \(2i\)
    \(2i+1\)

Correct!

Incorrect. The correct answer was: \(2i+1\)

18. What is the runtime of a breadth first search algorithm that traverses through a graph represented with an adjacency list?

    \(O(|V|)\)
    \(O(|V|^2)\)
    \(O(|V|+|E|)\)
    \(O(|V|*|E|)\)

Correct!

Incorrect. The correct answer was: \(O(|V|+|E|)\)

19. What is the maximum load factor that open addressing can support (in a hash table)?

    0.7
    1
    2
    Open addressing can support any load factor.

Correct!

Incorrect. The correct answer was: 1

20. Which of the following statements about a trie is INCORRECT?

    A trie is a binary tree.
    A trie is a "prefix tree."
    If word length is bounded, there is constant access to each word in the trie.
    In a trie, we need to mark where words end.

Correct!

Incorrect. The correct answer was: A trie is a binary tree.
