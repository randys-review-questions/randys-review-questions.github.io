Randy's Review Questions

CS 167: Computational Biology

Based on content taught by Donna Slonim in Spring 2022.

1. The Needleman-Wunsch sequence alignment algorithm is an example of a(n)...

    iterative algorithm.
    greedy algorithm.
    dynamic programming algorithm.
    naive algorithm.

Correct!

Incorrect. The correct answer was: dynamic programming algorithm.

2. Consider the following scoring scheme for sequence alignment. This scoring scheme would be appropriate for...

Match score = -3
Mismatch score = 1
Gap score = 2
    global alignment.
    local alignment.
    All of the above
    None of the above

Correct!

Incorrect. The correct answer was: global alignment.

3. One parameter of the BLAST algorithm is a scoring threshold \(T\) (where "neighbor words" of a \(w\)-mer is defined as all sequences that align with the \(w\)-mer with a score of at least \(T\)). Lowering the \(T\) value would increase the _____, which would _____.

    false positive rate, sacrifice accuracy for speed
    false positive rate, sacrifice speed for accuracy
    false negative rate, sacrifice accuracy for speed
    false negative rate, sacrifice speed for accuracy

Correct!

Incorrect. The correct answer was: false positive rate, sacrifice speed for accuracy

4. When computing a best MSA using the bounded dynamic programming approach, which of the following would be an upper bound on the similarity score?

If there are multiple correct answers, select the one that would be the tightest upper bound.

    The optimum alignment score of the best-scoring pair in the alignment
    The optimum alignment score of the worst-scoring pair in the alignment
    The sum of the optimum alignment scores of all pairs in the alignment
    The sum-of-pairs score of any arbitrary MSA.

Correct!

Incorrect. The correct answer was: The sum of the optimum alignment scores of all pairs in the alignment

5. Consider the following three sequences. Suppose that we want to align them using the star alignment method. Assuming that all matches are treated the same, all mismatches are treated the same, and there are never any gaps, which sequence would be at the "center" of the star?

Sequence 1: ATTCG
Sequence 2: ATACG
Sequence 3: ATTCC
    Sequence 1
    Sequence 2
    Sequence 3
    There is a tie, so the sequence would be selected arbitrarily.

Correct!

Incorrect. The correct answer was: Sequence 1

6. Suppose that at position 2 in a local MSA, there are the following nucleotides: A, A, A, and A. Assuming pseudocounts of 1, what is the frequency of A at position 2?

    50%
    62.50%
    87.50%
    100%

Correct!

Incorrect. The correct answer was: 62.50%

7. The compressive BLAST algorithm involves _____ preprocessing and _____ search.

    sublinear, sublinear
    linear, sublinear
    polynomial, linear
    exponential, linear

Correct!

Incorrect. The correct answer was: linear, sublinear

8. Suppose that there exists a branching node in a de Bruijn graph of \(k\)-mers. What does this imply?

    There exists an Eulerian path in the equivalent graph of \((k-1)\)-mers.
    There does not exist an Eulerian path in the equivalent graph of \((k-1)\)-mers.
    There exists an Eulerian path in the equivalent graph of \((k+1)\)-mers.
    There does not exist an Eulerian path in the equivalent graph of \((k+1)\)-mers.

Correct!

Incorrect. The correct answer was: There does not exist an Eulerian path in the equivalent graph of \((k-1)\)-mers.

9. Consider the following overlap graph. What is one possible result of executing greedy assembly on that graph?

    TACAGTTCAGA
    TACAGAGTTCA
    TACAGGTTCACAGA
    TACAGCAGAGTTCA

Correct!

Incorrect. The correct answer was: TACAGAGTTCA

10. Organisms that overall have a high GC content will have the most significant bias towards G and C in the _____ base of a codon.

    first
    second
    third
    None of the above

Correct!

Incorrect. The correct answer was: third

11. Which of the following algorithms finds the most likely state sequence when given a sequence of observable symbols and the specification of a Hidden Markov Model?

    Forward algorithm
    Viterbi algorithm
    Maximum Likelihood Estimation
    Baum-Welch algorithm

Correct!

Incorrect. The correct answer was: Viterbi algorithm

12. Suppose that in a given experiment, three people with sickle cell anemia had the following levels of expression of Gene 1: 7, 9, 11. Further suppose that three people without the disease had the following levels of expression of the same gene: 2, 4, 3. On average, there was a(n) _____ in expression of Gene 1 for those with sickle-cell anemia.

    3-fold increase
    6-fold increase
    3-fold decrease
    6-fold decrease

Correct!

Incorrect. The correct answer was: 3-fold increase

13. The molecular clock hypothesis holds for _____ trees, which satisfy the _____ condition.

    additive, 3-point
    additive, 4-point
    ultrametric, 3-point
    ultrametric, 4-point

Correct!

Incorrect. The correct answer was: ultrametric, 3-point

14. The problem of finding the most parsimonious tree when given an MSA can be solved in... (select the one that is most efficient)

    sublinear time.
    linear time.
    polynomial time.
    None of the above

Correct!

Incorrect. The correct answer was: None of the above

15. Anomaly detection is a method of classification that seeks to classify data points as "normal" or "anomalous." Such a classifier is trained with...

    normal data points only.
    anomalous data points only.
    labelled examples of normal and anomalous data points.
    unlabelled examples of normal and anomalous data points.

Correct!

Incorrect. The correct answer was: normal data points only.
