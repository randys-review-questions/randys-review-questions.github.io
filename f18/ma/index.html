Randy's Review Questions

Math Analysis

Based on content taught by John Redford in the 2018-2019 school year. Math Analysis is an accelerated form of pre-calculus.

1. When dividing variables with the same base, you _____ the exponents.

    Add
    Subtract
    Multiply
    Divide
    None of the above

Correct!

Incorrect. The correct answer was: Subtract

2. Which of these functions has an inverse?

    \(b(x)\), which burns \(x\)
    \(f(x)\), which freezes \(x\)
    \(c(x)\), which cooks \(x\)
    \(e(x)\), which eats \(x\)
    \(k(x)\), which kills \(x\)

Correct!

Incorrect. The correct answer was: \(f(x)\), which freezes \(x\)

3. \(\log(x)+\log(y)=?\)

    \(\log(x+y)\)
    \(\log(x-y)\)
    \(\log(x*y)\)
    \(\log(\frac{x}{y})\)
    \(\log(x^y)\)

Correct!

Incorrect. The correct answer was: \(\log(x*y)\)

4. The reciprocal of \(\cos(x)\) is equal to...

    \(\sin(x)\)
    \(\tan(x)\)
    \(\sec(x)\)
    \(\csc(x)\)
    \(\cot(x)\)

Correct!

Incorrect. The correct answer was: \(\sec(x)\)

5. \(\cot(x)+1=?\)

    \(\tan(x)\)
    \(\sec(x)\)
    \(\csc(x)\)
    \(\sec(x) \csc(x)\)
    It cannot be determined without more information

Correct!

Incorrect. The correct answer was: It cannot be determined without more information

6. What is the period of the following sinusoidal function: \(x=\cos{(\frac{2 \pi}{5}*t)}\)?

    \(2 \pi\)
    \(\frac{2 \pi}{5}\)
    \(\frac{5}{2} \pi\)
    \(\frac{5}{2 \pi}\)
    \(5\)

Correct!

Incorrect. The correct answer was: \(5\)

7. The ambiguous case describes a situation where...

    There are no possible triangles that fit the given criteria
    Given two side lengths and one angle, there are two possible triangles
    Given two angles and one side length, there are two possible triangles
    Given two side lengths and one angle, there are three possible triangles
    Given two angles and one side length, there are three possible triangles

Correct!

Incorrect. The correct answer was: Given two side lengths and one angle, there are two possible triangles

8. In a default parametric equation, there exists __ input(s) and __ output(s).

    1, 0
    1, 1
    1, 2
    2, 1
    2, 2

Correct!

Incorrect. The correct answer was: 1, 2

9. Which expression results in a finite geometric series?

    \(A_1 + (n-1) d\)
    \(\frac{n}{2} (A_1 + A_n)\)
    \(A_1 * r^{n-1}\)
    \(A_1 * \frac{1 - r^n}{1 - r}\)
    \(\frac{A_1}{1 - r}\)

Correct!

Incorrect. The correct answer was: \(A_1 * \frac{1 - r^n}{1 - r}\)

10. The limit as x approaches 1 is...

    The output where x equals 1
    The output value approached when x gets progressively closer to 1
    The output where x equals the reciprocal of 1
    Both the first option and the second option
    None of the above

Correct!

Incorrect. The correct answer was: The output value approached when x gets progressively closer to 1
